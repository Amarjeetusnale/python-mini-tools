# 🔐 Password Generator (Python)

## 📌 Description

A simple and secure password generator built using Python.
It generates strong, random passwords with customizable length and character options.

This project demonstrates Python fundamentals like randomness, string handling, and CLI interaction.

---

## ✨ Features

* Generate strong random passwords
* Customizable password length
* Includes:

  * Uppercase & lowercase letters
  * Digits (0–9)
  * Special characters (!, @, #, etc.)
* Ensures at least one character from each category
* Simple command-line interface

