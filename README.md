# 🔐 Password Generator (Python)

## 📌 Description
This is a simple and secure password generator built using Python.  
It allows users to generate strong and random passwords based on their preferences such as length, inclusion of digits, and special characters.

This project is beginner-friendly and demonstrates the use of Python modules like `random` and `string`.

---

## ✨ Features
- Generate random passwords of custom length  
- Option to include:
  - Uppercase & lowercase letters  
  - Digits (0–9)  
  - Special characters (!, @, #, etc.)  
- Strong password generation with at least one character from each category  
- Simple and easy-to-use command line interface
